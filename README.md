# 🔥 C Language Programs

Welcome to my C Programming repository!

This repository contains C language programs, examples, and practice exercises created during my learning journey. It covers fundamental programming concepts and helps build a strong foundation in problem-solving and software development.

## 📚 Topics Covered

- Introduction to C
- Variables & Data Types
- Operators
- Input/Output Functions
- Conditional Statements
- Loops
- Functions
- Arrays
- Strings
- Pointers
- Structures & Unions
- File Handling
- Dynamic Memory Allocation
- Pattern Programs
- Basic Problem Solving

## 🎯 Goals

- Learn C Programming Fundamentals
- Improve Logic Building Skills
- Practice Problem Solving
- Prepare for Academic and Technical Interviews

## 🛠️ Requirements

- GCC Compiler
- Code::Blocks / VS Code / Turbo C
- Basic Knowledge of Programming

## 🚀 How to Run

1. Clone the repository
2. Open the program in your preferred IDE
3. Compile and run

```bash
gcc program.c -o program
./program
```
